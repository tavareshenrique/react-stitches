# :construction_worker: Running

```bash
# Clone the Repository
$ git@github.com:tavareshenrique/react-stitches.git
```

```bash
# Download the dependencies
$ yarn
```

```bash
# Running
$ yarn start
```

Go to <http://localhost:3000> to see the result.
